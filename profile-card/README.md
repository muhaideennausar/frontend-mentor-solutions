- Semantic HTML5 markup
- CSS3 custom properties (CSS variables)
- Flexbox layout
- Mobile-first workflow
- Responsive images & media queries

## What I learned

- Positioning two background SVGs using `vw` and `vh` units to match the design precisely while keeping them responsive.
- Overlapping the avatar image using negative margin and a white border to create a smooth visual break from the card banner.
- Using CSS variables to centralise the colour palette, making the stylesheet easier to maintain and update.

## Continued development

- Add a dark mode toggle to practise theme switching with CSS custom properties.
- Improve accessibility by adding ARIA labels for the stats and ensuring all interactive elements have clear focus states.

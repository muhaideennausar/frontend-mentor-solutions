## What I learned

- **Dynamic Layouts with CSS Variables:** Used CSS `calc()` combined with inline custom variables (`style="--index: 0"`) to easily stagger the rating cards without writing repetitive margin classes.
- **Efficient SVGs:** Implemented inline SVGs using `<symbol>` and `<use>` tags to reuse the star icon efficiently, keeping the HTML clean and avoiding multiple external image requests.
- **Multiple Backgrounds:** Managed multiple background SVG patterns simultaneously, adjusting their positions and scaling across desktop and mobile breakpoints.

## Continued development

- **Accessibility:** Improve screen reader accessibility by adding descriptive `aria-labels` for the star rating sections.
- **Animations:** Add subtle entrance animations for the rating and review cards to enhance the user experience on load.

# 🛍️ Product Preview Card Component

A beautifully crafted **responsive product preview card** built using pure **HTML & CSS**. This project is inspired by a challenge from Frontend Mentor and focuses on clean UI design, responsiveness, and modern styling techniques.

---

## ✨ Preview

A sleek and modern product card showcasing:

- Product image (responsive for mobile & desktop)
- Category label
- Product title & description
- Pricing with discount highlight
- Add to cart button with icon

---

## 🚀 Features

- 📱 **Fully Responsive Design**
- 🎨 **Modern UI with Clean Typography**
- ⚡ **Lightweight & Fast (No JavaScript)**
- 🧩 **Component-Based Structure**
- 🎯 **Pixel-Perfect Layout**

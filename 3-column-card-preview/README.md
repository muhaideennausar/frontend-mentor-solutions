## Built with

- Semantic HTML5 markup
- CSS3 custom properties (CSS variables)
- Flexbox / Grid layout
- Mobile-first workflow
- Responsive images & media queries

## What I learned

- Using **relative units (em)** for spacing and typography to improve scalability across different screen sizes.
- Implementing **BEM naming convention** (`.cards-container__card-item`) to keep CSS modular and maintainable.
- Creating **hover and touch states** for buttons using `@media (hover: hover)` and `@media (hover: none)` to ensure a good experience on both desktop and mobile devices.
- Applying **border-radius** to the outer cards to achieve curved corners on desktop, then overriding them on mobile for a stacked column layout.

## Continued development

- Add **ARIA labels** and improve keyboard navigation for fully accessible interactive elements.
- Convert the layout from Flexbox to **CSS Grid** to experiment with alignment and gap properties.

# Order Summary Challenge

A clean and responsive solution to the **Frontend Mentor Order Summary Challenge** by **@muhaideennausar**.

## Live Demo

- **Live Site:** https://muhaideennausar.github.io/frontend-mentor-solutions/order-summary-challenge/
- **GitHub Repository:** https://github.com/muhaideennausar/frontend-mentor-solutions/tree/main/order-summary-challenge

## Overview

This project is a desktop-and-mobile friendly order summary card built with semantic HTML and modern CSS. It presents a subscription plan, pricing details, and action buttons in a visually polished layout. The page uses the **Red Hat Display** font, CSS custom properties for color management, and responsive media queries to adapt the layout for smaller screens. The HTML structure includes a hero illustration, plan information, and primary/secondary actions, while the stylesheet defines the card layout, spacing, shadows, hover states, and mobile background switching.

## Features

- Responsive card layout for desktop and mobile
- Clean typography with **Red Hat Display**
- Carefully tuned spacing, shadows, and rounded corners
- Hover and active states for interactive elements
- Mobile-specific background switching
- Simple, accessible, and beginner-friendly structure

## Built With

- HTML5
- CSS3
- Flexbox
- CSS variables
- Responsive media queries

```text

├── assets/
│   ├── icon-music.svg
│   ├── illustration-hero.svg
│   ├── pattern-background-desktop.svg
│   └── pattern-background-mobile.svg
├── screenshots/
│   ├── desktop-screenshot.png
│   └── mobile-screenshot.png
├── index.html
```

The project uses a single `index.html` file for the markup and a dedicated `style.css` file for all styling. The HTML links the stylesheet and Google Fonts, and the CSS handles layout, colors, responsiveness, and interaction states.

## Getting Started

To run the project locally:

1. Clone the repository

   ```bash
   git clone https://github.com/muhaideennausar/frontend-mentor-solutions.git
   ```

2. Navigate to the challenge folder

   ```bash
   cd frontend-mentor-solutions/order-summary-challenge
   ```

3. Open `index.html` in your browser
